                        সচরাচর জিজ্ঞাস্য (FAQ)
                        Frequently Asked Questions
                    

                    
                    

                        

                            এটি কি বাস্তব SMS পাঠায়?
                            
                        

                        

                            Does this send real SMS?
                            
                        

                        

                            না, এটি একটি শিক্ষামূলক ডেমো প্রকল্প। এটি বাস্তব SMS পাঠায় না, কিন্তু বাস্তব API কলের অনুকরণ করে। শিক্ষার্থীরা কিভাবে SMS API ইন্টিগ্রেশন কাজ করে তা বুঝতে এটি ব্যবহার করতে পারে।
                        

                        

                            No, this is an educational demo project. It doesn't send real SMS but simulates real API calls. Students can use it to understand how SMS API integration works.
                        

                    

                    
                    

                        

                            বাস্তব API ব্যবহার করতে কী করব?
                            
                        

                        

                            How to use a real API?
                            
                        

                        

                            ১. একটি বৈধ SMS সার্ভিস প্রোভাইডার থেকে API কী নিন (যেমন: Twilio, ClickSend, স্থানীয় প্রোভাইডার)

                            ২. API কীটি ব্যাকএন্ড সার্ভারে সুরক্ষিতভাবে সংরক্ষণ করুন

                            ৩. ফ্রন্টএন্ড থেকে সরাসরি API কী পাঠাবেন না

                            ৪. আপনার সার্ভার থেকে SMS API কল করুন
                        

                        

                            1. Get an API key from a legitimate SMS service provider (e.g., Twilio, ClickSend, local providers)

                            2. Securely store the API key in your backend server

                            3. Never send API key directly from frontend

                            4. Make SMS API calls from your server
                        

                    

                    
                    

                        

                            কেন ১ মিনিটের রেট লিমিট?
                            
                        

                        

                            Why 1-minute rate limit?
                            
                        

                        

                            এটি অপব্যবহার রোধ করার জন্য একটি শিক্ষামূলক সুরক্ষা বৈশিষ্ট্য। বাস্তব API গুলিতে সাধারণত রেট লিমিট থাকে (প্রতি মিনিটে X অনুরোধ)। এটি শিক্ষার্থীদের রেট লিমিটিং কনসেপ্ট বুঝতে সাহায্য করে।
                        

                        

                            This is an educational safety feature to prevent misuse. Real APIs typically have rate limits (X requests per minute). This helps students understand rate limiting concepts.
                        

                    

                    
                    

                        

                            আমার ডেটা কিভাবে সংরক্ষিত হয়?
                            
                        

                        

                            How is my data stored?
                            
                        

                        

                            সমস্ত ডেটা শুধুমাত্র আপনার ব্রাউজারের লোকাল স্টোরেজে সংরক্ষিত হয়। সার্ভারে কোনো ডেটা পাঠানো হয় না। মোবাইল নম্বরগুলি ইতিহাসে মাস্ক করা দেখানো হয় (017******78)। আপনি যখন ব্রাউজার ক্লিয়ার করেন তখন ডেটা মুছে যায়।
                        

                        

                            All data is stored only in your browser's local storage. No data is sent to any server. Phone numbers are masked in history display (017******78). Data is cleared when you clear your browser.
